# Power BI Dashboard – KPI & Sales Performance

## Project Overview
This Power BI project involves the creation of an interactive dashboard to visualize KPIs, regional sales distribution, and performance metrics. It demonstrates the use of DAX formulas, slicers, and filters to enable dynamic data exploration.

## Key Features
- **Interactive dashboard** for real-time sales tracking
- **DAX formulas** used for custom measures and KPIs
- **Slicers and filters** for user-driven analysis (e.g., region, category, time)
- Visualizations include bar charts, line graphs, cards, and tables
- Clear **visual storytelling** to support business decision-making

## Tools & Techniques Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Filters, Slicers, Visual Interactions
- Card, Matrix, Bar & Line Chart Visuals
- Data Modeling & Relationship Building

## Sample Insights Delivered
- KPI tracking (sales, targets, growth)
- Region-wise and product-wise performance
- Time-series analysis of sales trends

## File Included
- `Dashboard4.pbix` – Full interactive dashboard with visuals and DAX measures
